# Check Session Workflow

## Purpose
Save current progress mid-session so nothing is lost.

## What to Save

1. **Current Progress** - What has been done this session
2. **Pending Items** - What's still in progress
3. **Blockers** - Any issues encountered
4. **Next Steps** - What to do next

## Steps

1. Review current conversation for progress
2. Update TASKS.md with completed items
3. Note any blockers or decisions made
4. Log to SESSION_LOG.md

## Output

```markdown
## Session Checkpoint - [timestamp]

### Progress
- [x] Item completed
- [/] Item in progress

### Blockers
- Issue 1

### Next Steps
- Step 1
```
