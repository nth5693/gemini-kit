# Start Session Workflow

## Purpose
Load context from previous session to continue work seamlessly.

## Files to Read

1. **SESSION_LOG.md** - Previous session notes
2. **TASKS.md** - Current task list
3. **IMPLEMENTATION_PLAN.md** - Active plan
4. **WORKFLOW.md** - Workflow context

## Steps

1. Read SESSION_LOG.md for previous progress
2. Read TASKS.md to check pending items
3. Read IMPLEMENTATION_PLAN.md for active plan
4. Summarize what was done and what's next

## Output

Provide:
- Summary of previous session
- Current task status
- Recommended next steps
